What am I currently working on?

I am currently working on learning more c#/asp.net and working on my switchterm application. I am trying to build the foundation for an application generator and a command generator.

Why build this?

There are solutions for running containers and applications using big systems like kubernetes and docker. I believe these things can and are just functions. Why not put the idea into code and generate functions. You can generate bash commands if you need to start things that way.

Why reinvent the wheel?

Running and stopping applications shouldn't need to be a whole framework, gui, or anything. It is just code and just a function. Why not generate it?
